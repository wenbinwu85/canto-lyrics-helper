歌词助手

最近迷上写歌词，发现写广东歌词真的很难， 所以些了个助手帮忙。

主要功能：
- 粤拼搜查
- 中文词典
- 同音字搜查
- 从 Mojim 下载歌词
- 简单用户界面 (參考 screenshot)

要求：
- Python 3.6+
- Requests
- BeautifulSoup4
- html5lib
- wxpython phoenix
